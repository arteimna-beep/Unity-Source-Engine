# Unity-Source-Engine
Replication of the Source Engine in unity engine!

Hello, this package is not finished and its can be difficult for people
who don't know anything about baking light and stuff.

Here i gonna tell you how to control thus package and make your maps looks like in source.

--IMPORTANT STUFF--

- Don't use unity default skybox, use sky boxes inside the package.

- Change something inside baking if you know what you doing only.

- Change Map resolution depending on how big your map is, to not get 
  terrible shadows.
